* This repo is for the KSU Social Good Hackathon 2026.

* We are participating in the Assurant track.

* Our project is a Contract-Aware AI Control Plane.

* It is a system that can be used to route AI requests to different models based on the customer's contract and SLA.

* ASSURANT_CONTRACT_AWARE_AI_CONTROL_PLANE_SPEC.md contains the full specification for the project.

* assurant_hackathon_2026.pdf contains the hackathon track information.
